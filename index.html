Planogram AI

Smarter retail layout planning. Powered by AI. Managed with precision.

📦 Features

🧠 AI Layouts
Automatically suggest optimized shelf arrangements.
📅 Rollout Calendar
Visualize planogram schedules with fullcalendar integration.
🔐 Role-Based Access
Manage permissions for merchandisers, managers, and admins.
📊 Reporting Tools
Export layout activity as CSV or PDF.
📬 Email Reminders
Get notifications on scheduled or pending layouts.
📝 Activity Logs
Track layout history, changes, and approvals.

🚀 Getting Started

git clone https://github.com/your-org/planogram-ai.git
cd planogram-ai
npm install
npm run dev

Set up your environment using .env and connect to your Supabase project.
